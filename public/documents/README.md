# Downloadable Documents

PDF guides and resources for Lone Mountain buyers and sellers.

## Recommended Documents

### For Buyers
- `lone-mountain-buyer-guide.pdf` - Complete guide to buying in Lone Mountain
- `lone-mountain-neighborhood-guide.pdf` - Area amenities, schools, and lifestyle
- `relocation-checklist.pdf` - Moving to Las Vegas checklist

### For Sellers
- `lone-mountain-seller-guide.pdf` - How to sell your Lone Mountain home
- `home-prep-checklist.pdf` - Preparing your home for sale
- `pricing-strategy-guide.pdf` - Understanding market pricing

### General
- `market-report-current.pdf` - Latest Lone Mountain market statistics
- `school-district-info.pdf` - Local school information

## Creating PDF Guides

Use Canva, Adobe InDesign, or similar tools to create professional PDFs.

### Branding Guidelines
- Primary Color: Navy Blue (#1a365d)
- Accent Color: Gold (#d69e2e)
- Font: Clean serif for headings, sans-serif for body
- Include Dr. Jan Duffy contact info and BHHS branding
- Add QR code linking back to website

## Usage on Website

Link to documents from relevant pages:

```jsx
<a
```
