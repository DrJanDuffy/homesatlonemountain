# Images Directory

This folder contains all site images for Homes at Lone Mountain.

## Folder Structure

```
images/
├── agent/          # Agent headshot and professional photos
│   └── jan-duffy.jpg   # Dr. Jan Duffy professional headshot (recommended: 800x800)
├── blog/           # Blog post featured images (recommended: 1200x630)
│   └── market-update-q1-2024.jpg
├── properties/     # Property listing images (recommended: 1200x800)
│   ├── luxury-home-1.jpg
│   ├── luxury-home-2.jpg
│   └── luxury-home-3.jpg
└── logo.png        # Site logo (recommended: 400x100 or square)
```

## Image Guidelines

- **OG/Social images**: 1200x630px (already at `/public/og-image.jpg`)
- **Agent headshot**: 800x800px square, professional quality
- **Blog images**: 1200x630px landscape
- **Property images**: 1200x800px landscape, high quality
- **Logo**: PNG with transparency, ideally 400px wide
